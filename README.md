### Run Locally

To run **alck-portfolio** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/LoickAdjiwanou/alck-portfolio/
```

Windows:

```bash
git clone https://github.com/LoickAdjiwanou/alck-portfolio/
```
